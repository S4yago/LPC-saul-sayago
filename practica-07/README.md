# FTP

Script de bash que busca archivos en un servidor FTP, para
después descargarlo.

**Requisitos:**<br>

<table>
<tr>
    <td>Arch</td>
    <td><code>sudo pacman -S lftp</code></td>
</tr>
<tr>
    <td>Debian/Ubuntu</td>
    <td><code>sudo apt install lftp</code></td>
</tr>
</table>
